ココチモ（cococimo）の特集

”心地いい”毎日を過ごしていただくために、お客様おひとりおひとりの生活に寄り添う商品をご提供する「ココチモ―ＣＯＣＯＣＩＭＯ」。その中でも特におすすめのものをご紹介いたします。季節に嬉しいアイテム、世界で話題の逸品、手にするだけで気持ちが明るくなるグッズ等々――。暮らしに潤いと彩りを添える特別な一品を、どうぞご覧ください。

「おいしい食卓・年末号」より、おすすめのディナーをご紹介します。
簡単なのに見栄えも味も本格的で素敵なディナーができあがります。
ご夫婦で、ご家族で、お友達と、心弾むクリスマスの食卓をどうぞ！

キャスターの草野仁さんに「ココチモオリジナル　カシオ電子辞書」について
お聞きしました。草野さんならではの視点で言葉について、学ぶことについての
想いが語られています。ぜひ、ご覧ください。

触って楽しめる「日本立体大地図」の制作担当者と“地図の神様”の異名を持つ
今尾恵介氏による座談会。地図好き必見の面白いお話がいっぱいです。

「おいしい食卓・秋号」に掲載の食材を使って、本格的な和朝食を再現
しました。わずか10分で出来る時短朝食です。栄養士のアドバイスも
掲載。あなたの朝食づくりにお役立てください。

好評をいただいているココチモオリジナル電子辞書。お客様にとって使いやすい
電子辞書とは何か？　電子辞書メーカーシェアNo.1のカシオと共同開発した
電子辞書。バイヤー坂口の奮闘とともにその誕生秘話をお伝えします。

「防災について考える」特集の第2弾です。今回は災害時にペットと避難する際、
気を付けたいことや、用意したいものなどについて防災危機管理アドバイザーの
山村武彦氏にお話を伺いました。

ココチモオリジナルのにんじんジュースが誕生しました。
にんじんジュースの産地や栄養についてなどココチモのこだわりや
商品の魅力を余すところなくお伝えします。

豊かな自然に囲まれた山形県・高畠町にあるスモークハウスファインを
訪ねました。無添加のウインナーやベーコンが誕生するまでの秘話を
名物社長、片平氏が語ります。

「おいしい食卓・秋号」は健康・作り手の想い・味わいに
新たに国産と化学調味料不使用のこだわりが加わりました。
実りの秋を存分に感じていただける商品ラインナップです。

もうすぐ防災の日。防災・危機管理アドバイザーの山村武彦氏に、
2016年4月に起きた熊本地震の実例を交え災害時の屋外での避難生活について
お話をうかがいました。日頃の防災対策にお役立てください。

まんが日本昔ばなしで有名な「耳なし芳一」や「おいてけ堀」など
有名な怪談話をご紹介。
こわ〜いむかし話で、体もココロも涼しいひとときを過ごしてみませんか。

ココチモから新しい食のブランド「おいしい食卓」が誕生しました。
健康・作り手の想い・味わいにこだわったものを厳選。
夏を元気に過ごす旬の味覚をお楽しみください。

音楽ライター・近藤正義氏にティアック多機能オーディオに寄せて、
今こそレコードで聞きたいおすすめの名曲をご紹介していただきました。
あなたのお好きな曲もきっとあるはず。

ココチモサイトでも人気の「まんが日本むかし話」。
今回は好きなお話や子供に見せたいお話などいろんな切り口で、アンケートを取り
そのランキングを発表しています。

だれでもがご存知の昔話「花咲か爺さん」。
子供はもちろんのこと大人になってから改めて見てみるとまた違う感動が・・・。
大人だからこそ分かる「花咲か爺さん」の魅力をご紹介します。

2015年3月に定期運用から外れる北斗星。一時代を築いた豪華寝台列車
北斗星に寄せて、アナウンサーの堺正幸さんとライターの池口英司さんが
北斗星の旅を楽しみながらその魅力について語らいます。

新潟県燕市の工房アイザワを訪ね、作り手の想いを取材しました。
生産現場にも訪れ、職人の技を体感！
さらに料理研究家によるお弁当講座もご紹介しています。

実際はエクアドルが本場なのだとか。
夏にぴったりのおしゃれでイキなパナマ帽子の魅力に迫ります。
